# cs81-module3a-github
CS 81 Module 3A assignment - Learning Git version control and GitHub by creating a JavaScript function journal

## Functions in journal.js

### 1. greet(name)
Greets a user by name and returns a personalized hello message.
- **Parameter**: `name` (string) - The name of the person to greet
- **Returns**: A greeting string

### 2. isEven(number)
Checks if a given number is even.
- **Parameter**: `number` (number) - The number to check
- **Returns**: `true` if the number is even, `false` otherwise

### 3. add(a, b)
Calculates the sum of two numbers.
- **Parameters**: `a` (number), `b` (number) - The numbers to add
- **Returns**: The sum of the two numbers

### 4. celsiusToFahrenheit(celsius)
Converts temperature from Celsius to Fahrenheit.
- **Parameter**: `celsius` (number) - Temperature in Celsius
- **Returns**: Temperature in Fahrenheit

### 5. findMax(num1, num2)
Finds the maximum value between two numbers.
- **Parameters**: `num1` (number), `num2` (number) - The numbers to compare
- **Returns**: The larger of the two numbers
